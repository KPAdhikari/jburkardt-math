MHD_CONTROL_MOVIE
Animate MHD Data

MHD_CONTROL_MOVIE is a pair of Mathematica routines which display the magnetic field or velocity field, using data from a particular run of a magnetic hydrodynamic control problem for a fluid flow in a 2D region.

Related Data and Programs:

The program used to generate the data is available in MHD_CONTROL.

The data files used for this particular animation are available in the MHD_CONTROL dataset directory.

Source Code:

  • mag_control_movie.nb, a script which reads the magnetic field data and displays it, and can animate it.
  • vel_control_movie.nb, a script which reads the velocity field data and displays it, and can animate it.
